
# Full Stack Chat Application — MERN + Socket.io

![Chat App Screenshot](https://github.com/amena21/chatApplication/blob/main/loginpage.png)


A real‑time chat application built using the MERN stack (MongoDB, Express, React, Node.js) with Socket.io for instant messaging. This project includes user authentication, private chats, and a clean, responsive UI.
Deployed using Render (backend) and Vercel (frontend).



## Features

- Real-time messaging using Socket.io
- User authentication (JWT + bcrypt)
- Private one‑to‑one chat
- Online/offline user status
- MongoDB database for storing users & messages
- Responsive UI built with React + Tailwind
- Production-ready deployment on Render + Vercel

## Tech Stack

**Client:** React, Context API, Tailwind CSS

**Server:** Node, Express

**Real-time:** Socket.io

**Database:** MongoDB 

**Special Services:** Cloudinary handles image uploads, Resend sends automated welcome emails, and
Arcjet provides security features like rate limiting and bot detection

**Deployment:** Render (API) + Vercel (UI)
